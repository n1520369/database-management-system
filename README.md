# Database Management System

## Overview

A Python-based database management system using SQLite. This project demonstrates CRUD operations, database connectivity, and record management.

## Features

- Create employee records
- Read employee records
- SQLite database integration
- Data storage and retrieval

## Technologies Used

- Python
- SQLite

## Database Schema

Employee Table

- ID
- Name
- Department
- Salary

## Installation

No external dependencies required.

## Run

python database_manager.py

## Sample Output

(1, 'John', 'IT', 60000)
